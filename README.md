# 🩺 API – Cartão de Vacinação

Este projeto implementa uma API REST para gerenciamento de informações de vacinação, incluindo pacientes, imunizações, vacinas e estatísticas relacionadas.  
O objetivo é oferecer um backend simples, direto e funcional para manipulação dos dados de um cartão de vacinação digital.

---

## Tecnologias Utilizadas

- Java 21  
- Spring Boot  
- Spring Web  
- Spring Data JPA  
- MySQL (produção)  
- H2 Database (testes)  
- Maven  
- Thunder Client (testes manuais)
